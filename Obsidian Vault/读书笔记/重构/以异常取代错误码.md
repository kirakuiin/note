> [!summary] 概要
> 某个函数返回一个特定的代码，用以代表某种错误情况。此时应该改用异常。

> [!summary] 动机
> 异常将普通程序运行逻辑和错误区分开来，而这让程序更加容易被理解。使用返回值会混淆这两者。

> [!summary] 做法
> 1. 新建一个函数，使用异常来表达错误情况，将旧函数的代码复制到其中并作出一定调整。
> 2. 修改旧函数，使其调用新函数。
> 3. 测试。
> 4. 逐一修改旧函数的调用者，使其调用新函数。
> 5. 删除旧函数。

> [!example] 范例
> 重构前：
> ```python
> def Foo():
> 	...
> 	if error:
> 		return 1
> 	else:
> 		return 0
> ```
> 重构后：
> ```python
> def Foo():
> 	...
> 	if error:
> 		raise Exception()
> ```
